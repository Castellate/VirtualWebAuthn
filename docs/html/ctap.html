from_bytes(data: bytes, without_id=False)[source]¶

Loads a credential source from bytes. This should be called when reloading a credential source from storage or after decrypting a non-resident key.

without_id is used to indicate whether the id attribute should be set, in the case of a non-resident key this should be True due to the id being the encrypted contents of this.

Parameters:
  • data (bytes) – bytes to load
  • without_id (bool, optional) – True to exclude the ID (non-resident
  • False to set the ID. Defaults to False. (key),) –

Constructs a JSON object containing the data associated with this credential source and encodes the JSON String as bytes. This should be used for storing the credential source to disk or when the credential source is going to be wrapped and set on the server, for example, a non-resident key.
